Heurísticas de Nielsen aplicadas
Status: a interface deve sempre informar o usuário sobre o que está acontecendo, através de feedbacks apropriados e em tempo razoável.
Prevenção de Erros: devem garantir que o usuário não erre na execução de algo, utilizando mensagens de erro ou mesmo designs que evitam que erros ocorram (ex: confirmar antes de excluir).
Estética: busca apresentar uma interface que contenha o essencial para o entendimento do usuário.
